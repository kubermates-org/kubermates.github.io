  designed with security, resilience, and enterprise scalability at its core. When
  paired with SQL Server 2025, customers benefit from: Post-quantum cryptography readiness
  and integrated hardware security support Operational simplicity with built-in AI-assisted
  management through Red Hat Lightspeed, reducing administrative overhead Hybrid cloud
  readiness optimized for on-premises, cloud, and edge deployments Enterprise reliability
  and compliance, trusted by organizations worldwide This makes RHEL 10 an ideal foundation
  for both development and production SQL Server workloads. To install the production-ready
  GA version of SQL Server 2025 on Red Hat Enterprise Linux 10, it''s important to
  update your repository configuration by switching from mssql-server-preview. repo
  to mssql-server-2025. repo. Continuing to use the preview repository may result
  in installing pre-release builds that are not intended for production workloads.
  See RHEL: Install SQL Server on Linux for installation instructions. mssql-server-preview.'
