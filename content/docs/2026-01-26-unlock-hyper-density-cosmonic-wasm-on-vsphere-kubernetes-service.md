  DBaaS: Repatriation and Cost Control In the rapidly evolving landscape of platform
  engineering, the shift from heavy virtual machines to containers was just the first
  step. Today, we are witnessing the next leap forward: WebAssembly (Wasm). Cosmonic,
  built on the foundations of the CNCF incubating project wasmCloud , is leading the
  charge with Cosmonic Control. By integrating this with the vSphere Kubernetes Service
  (VKS) within VMware Cloud Foundation (VCF) , enterprises can create a “Universal
  Golden Path” that blends the ironclad reliability of vSphere, the flexibility of
  Kubernetes, and the extreme efficiency of Wasm. Cosmonic Control is an enterprise-grade
  control plane for wasmCloud. It allows developers to build applications as sandboxed
  WebAssembly binaries with explicitly enabled capabilities like HTTP or messaging.
  Unlike containers, which package an entire OS user-space, Wasm components are platform-agnostic
  binaries that run in a tiny, secure sandbox.'
