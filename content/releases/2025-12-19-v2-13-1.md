  v2.13.1 is the latest patch release of Rancher. This is a Community version release
  that introduces maintenance updates and bug fixes. For more information on new features
  in the general minor release see the v2.13.0 release notes. Fixed an issue where
  in air-gapped environments when attempting to access the legacy v3 API-UI page directly
  via the host URL ( <RANCHER-SERVER-URL>/v3 ), the page fails to load and displays
  a blank page. <RANCHER-SERVER-URL>/v3 Fixed an issue with cluster provisioning not

  Custom or Node-Driver) with IPv6 CIDRs, the cluster would become stuck in a provisioning
  state and does not become Active.'
